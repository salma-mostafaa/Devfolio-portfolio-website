# 🌐 DevFolio – Personal Portfolio Website

A modern, responsive personal portfolio website built using HTML, CSS, JavaScript, and Bootstrap.  
It showcases my skills, projects, services, blog posts, and contact information in a clean UI.

---

## ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Sticky navbar with active section highlighting
- Hero section with animated typing effect
- About section with skills progress bars
- Services section with cards
- Portfolio gallery with hover zoom effects
- Testimonials carousel
- Blog section with preview cards
- Contact form UI + social media links
- Built using Bootstrap 5 + custom CSS

---

## 🛠️ Built With

- HTML5  
- CSS3  
- JavaScript   
- Bootstrap 5  
- Bootstrap Icons
